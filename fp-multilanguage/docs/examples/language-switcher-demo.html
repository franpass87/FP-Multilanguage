🌐 FP Multilanguage - Language Switcher Demo

1. Stile Inline - Con Bandierine

Il modo più comune per mostrare il selettore di lingua:

🇮🇹 Italiano / 🇺🇸 English
[fp_lang_switcher style="inline" show_flags="1"]

2. Stile Inline - Senza Bandierine

Versione minimalista solo con il testo:

Italiano / English
[fp_lang_switcher style="inline" show_flags="0"]

3. Stile Dropdown - Con Bandierine

Ideale quando hai poco spazio disponibile:

[fp_lang_switcher style="dropdown" show_flags="1"]

📋 Caratteristiche Principali

✨ Responsive Design

Si adatta automaticamente a tutti i dispositivi, con font-size maggiori su mobile.

🌙 Dark Mode

Supporto nativo per il dark mode, si attiva automaticamente.

♿ Accessibilità

Supporto completo per screen reader, navigazione da tastiera e ARIA labels.

🎨 Personalizzabile

Puoi facilmente sovrascrivere gli stili CSS nel tuo tema.

⚡ Leggero

CSS minimalista (~5KB) senza dipendenze JavaScript (tranne dropdown).

🔗 SEO Friendly

Link con rel="nofollow" per non sprecare crawl budget.

🎯 Come Usarlo

Metodo 1: Widget WordPress

1. Vai in Aspetto → Widget 2. Cerca "Selettore Lingua FP" 3. Trascina nella sidebar desiderata 4. Configura le opzioni

Metodo 2: Shortcode

// In qualsiasi post/pagina [fp_lang_switcher style="inline" show_flags="1"] // Nel codice PHP <?php echo do_shortcode('[fp_lang_switcher style="inline" show_flags="1"]'); ?>

Metodo 3: Codice PHP diretto

<?php if ( function_exists( 'FPML_Language' ) ) { $language = FPML_Language::instance(); echo $language->render_switcher( array( 'style' => 'inline', 'show_flags' => '1', ) ); } ?>

🎨 Personalizzazione CSS

Esempio di come personalizzare i colori nel tuo tema:

/* Cambia il colore della lingua corrente */ .fpml-switcher__item--current { background-color: #ff6b35; border-color: #ff6b35; } /* Cambia il colore al hover */ .fpml-switcher__item:hover { background-color: #f0f0f1; } /* Ingrandisci le bandierine */ .fpml-switcher__flag { font-size: 24px; }
